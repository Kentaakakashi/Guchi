# Pulse Lobby

Modern realtime multiplayer lobby system using:

- HTML
- CSS
- Vanilla JavaScript
- Firebase Realtime Database

## Features

- Realtime sync
- Create / Join lobby
- Host system
- Ready system
- Start game button
- Mobile friendly
- Glassmorphism UI
- Copy room code
- Leave lobby
- Max player limit

## Firebase Rules

Use Realtime Database rules:

{
  "rules": {
    ".read": true,
    ".write": true
  }
}
